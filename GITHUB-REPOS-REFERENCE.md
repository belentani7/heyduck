## 🔧 Solución para Imágenes en HTML

### Opción 1: URLs de Unsplash (Recomendado)
```html
<!-- Agregar en el <head> -->
<img src="https://images.unsplash.com/photo-1598488035139-bdbb2231ce04?w=1920&q=80" alt="Studio">
```

### Opción 2: Base64 embebido
```javascript
// Convertir imagen a base64 y usar directamente
const img = 'data:image/jpeg;base64,/9j/4AAQSkZJRg...';
```

### Opción 3: CSS background con fallback
```css
.hero-bg {
  background: url('images/hero.jpg') center/cover no-repeat;
  background-color: #080e08; /* fallback */
}
```
