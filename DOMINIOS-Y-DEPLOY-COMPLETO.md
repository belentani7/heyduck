## 📝 CÓMO OBTENER duck.is-a.dev

1. Crear cuenta en GitHub
2. Ir a `https://github.com/is-a-dev/register`
3. Fork del repositorio
4. Crear archivo `domains/duck.json`:
```json
{
  "description": "Duck - Musical Producer & Beatmaker",
  "name": "duck",
  "record": {
    "CNAME": "TU-USUARIO.github.io"
  }
}
```
5. Abrir Pull Request
6. Esperar aprobación (1-2 días)
7. ¡Tu dominio estará activo!
